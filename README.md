# Hand Gesture Controlled Galaxy

## Features
- Real-time hand gesture control (Python + OpenCV)
- Voice commands integration
- 3D galaxy visualization (Three.js)
- Multiplayer collaboration (Socket.IO)
- Gamification: missions & leaderboard
